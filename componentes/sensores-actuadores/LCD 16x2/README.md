# 🔌 Pantalla LCD 16x2 (16 columnas, 2 filas)

## 🧾 Descripción

Pantalla LCD de 16 columnas y 2 filas con protocolo I2C - letras blancas sobre fondo azul. Perfecto para mostrar los datos obtenidos por un sensor, o cualquier otro tipo de información relevante.
