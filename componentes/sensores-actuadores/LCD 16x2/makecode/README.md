# 🚧 En construcción

Para usar la pantalla LCD con la placa micro:bit y Reka:bit existen varias opciones.

1. Usar la extensión "Grove" del menú de extensiones de Makecode. Esta extensión no se puede usar en conjunto con la extensión Rekabit, por lo que para usar las características de esta placa (como los motores o los LEDs RGB) será necesario utilizar extensiones distintas para cada una de ellas.

2. Usa el siguiente URL en la barra de búsqueda de extensiones: [URL](\\wsl.localhost\Debian\home\tzarls\ecosistema)

En ambos casos aparecerán nuevos bloques que permitirán el uso de la pantalla LCD desde Makecode.

La pantalla LCD debe conectarse en el último conector Grove (del extremo derecho) de la placa Rekabit, pues es el único puerto que implementa el protocolo I2C que utliza este módulo para conectarse al micro:bit.
